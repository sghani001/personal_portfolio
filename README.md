## 🖥️ About the Project

This repository houses the source code for my personal digital portfolio. It serves as an interactive showcase of my full-stack engineering work, production background workflows, and overall design philosophy. 

The site is designed to deliver a highly immersive, dark-mode user experience that mirrors the sleek, high-contrast aesthetics of modern developer environments.

### ✨ Key Features
* **Interactive 3D Elements:** Immersive visual assets integrated directly into the layout.
* **Glassmorphic UI Design:** Clean boundaries, subtle neon glows, and professional dark gradients.
* **Responsive Layout:** Engineered from schema structures down to polished components for seamless cross-device compatibility.
* **Modular Architecture:** Clean division between content domains and active technical metrics.

---

## 🛠️ Architecture & Built With

The site leverages a streamlined, modern frontend stack focused on performance, modular component design, and smooth interactive graphics:

* **Core Framework:** `React.js` for handling dynamic components and state management.
* **3D Interactive Assets:** `Spline 3D` integration for fluid, real-time spatial visuals.
* **Hosting & Deployment:** Automated and securely deployed on `GitHub Pages`.
